🔐 Data Privacy & Security

Your scraped data belongs only to you.
	•	Every run creates its own isolated directory in secure Cloudflare R2 storage.
	•	Each dataset is stored under a unique, encrypted run ID — no shared folders.
	•	A private, time-limited link (24h by default) is generated for your download.
	•	No one — including the developer — can view, monitor, or retain your data.
	•	Once your presigned URL expires, access is permanently closed.

Your data is never shared, logged, or reused.
Every run is private, ephemeral, and fully under your control.
